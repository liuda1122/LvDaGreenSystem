## 图标规范

### 禁止使用 Emoji 作为 UI 图标

本项目已将所有 Emoji 图标替换为内联 SVG。所有 SVG 图标：
- 尺寸：16×16px（默认）
- 对齐：`vertical-align: middle`
- 颜色：使用 `currentColor`，继承父元素文字颜色
- 描边：`stroke-width="1.3"`（标准）/ `1.5`（强调）

### SVG 图标速查表

| 图标 | SVG 用途 | 之前 Emoji |
|------|----------|-----------|
| 剪贴板 | 信息块标题、表单、报告 | 📋 |
| 文本笔记 | 记录、备注、编辑 | 📝 |
| 勾选圆圈 | 成功状态、确认、启用 | ✅ |
| 灯泡 | 提示、建议、节能量 | 💡 |
| 工具 | 设置、开发备注 | 🔧 |
| 条形图 | 报表、统计、图表 | 📊 |
| 货币(¥) | 政策红利、金融产品 | 💰 |
| 靶心 | 优先级、目标、策略 | 🎯 |
| 图钉 | 标记、固定、分类标签 | 📌 |
| 地图 | 产业地图 | 🗺️ |
| 下载/保存 | 保存按钮 | 💾 |
| 文件 | 附件、文件预览 | 📄 |

### 新增图标规范

如需新增图标，遵循以下原则：
1. 优先使用内联 SVG，不要使用 Emoji
2. SVG 继承 `currentColor`，不硬编码颜色
3. 保持 16×16 或 20×20 viewBox
4. 添加到本文件的图标速查表

## 色彩规范

### 品牌色板（绿色金融主题）

本项目使用绿色品牌色系，**禁止使用紫色/蓝色/霓虹粉作为 UI 主色**。

| 颜色 | 色值 | 用途 |
|------|------|------|
| 主绿 | `var(--green-700)` | 主色调、强调 |
| 深绿 | `var(--green-800)` / `var(--green-900)` | hover、激活态 |
| 浅绿背景 | `var(--green-50)` / `var(--success-bg)` | 选中背景、成功提示 |
| 浅绿边框 | `var(--green-200)` / `var(--success-border)` | 边框、分隔 |
| 辅助色 | `var(--ink-800)` | 主文字 |
| 次级文字 | `var(--ink-400)` / `var(--ink-500)` | 描述文字 |
| 背景 | `var(--page-bg)` / `#f5f6fa` | 页面底色 |

### 禁止使用的 AI 典型配色

以下颜色已被识别为 AI 生成典型配色，**禁止在项目中使用**：
- 紫色系：#6366f1, #4f46e5, #8b5cf6, #7c3aed 等
- 霓虹粉：#d946ef, #c026d3, #e879f9 等
- 彩虹渐变（含 purple/violet/indigo）

## 错误提示规范

### 用户可见的错误信息

- 所有面向用户的错误提示必须使用**中文**
- 禁止将 `e.message`、`e.stack` 直接展示给用户
- 如未来添加 JS 错误处理，使用 `friendlyError()` 工具函数统一包装
